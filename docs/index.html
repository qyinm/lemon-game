Overview

Lemon Game is a simple puzzle game Chrome extension. We are committed to protecting your privacy. This policy explains what data we collect and how we use it.

Summary: Lemon Game does not collect, transmit, or share any personal data. All game data is stored locally on your device only.

Data Collection

Lemon Game does not collect any of the following:

  • Personal information (name, email, address, etc.)
  • Location data
  • Browsing history
  • User activity outside the game
  • Financial information
  • Any other personally identifiable information

Local Storage

Lemon Game uses Chrome's local storage API (chrome.storage.local) to save:

  • Your high score

This data is stored only on your device and is never transmitted to any external servers.

Permissions

Lemon Game requires the following permission:

  • storage: Used solely to save your high score locally on your device.

Third-Party Services

Lemon Game does not use any third-party services, analytics, or tracking tools.

Remote Code

Lemon Game does not load or execute any remote code. All code is bundled within the extension package.

Data Sharing

We do not sell, transfer, or share any user data with third parties.

Children's Privacy

Lemon Game does not collect any personal information from anyone, including children under 13 years of age.

Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page with an updated revision date.

Contact

If you have any questions about this Privacy Policy, please create an issue on our GitHub repository.
